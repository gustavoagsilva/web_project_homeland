# 🌍 Homeland

## 📌 Descrição do Projeto

O **Homeland** é uma página web desenvolvida para apresentar as cidades natais de diferentes pessoas, destacando suas culturas, histórias e características.

O projeto foi criado com foco na construção de um layout moderno e responsivo, permitindo que o site se adapte corretamente a diferentes tamanhos de tela, como desktops, tablets e smartphones.

A aplicação apresenta imagens, textos descritivos e links relacionados às cidades exibidas, proporcionando uma experiência visual agradável e organizada ao usuário.

---

## 🎯 Objetivo do Projeto

O objetivo deste projeto é praticar e consolidar conhecimentos em desenvolvimento **front-end**, especialmente na criação de layouts responsivos baseados em um design pré-definido no Figma.

Entre os principais objetivos estão:

- Desenvolver uma página web responsiva
- Trabalhar com **layouts modernos**
- Utilizar **boas práticas de organização de código**
- Aplicar a metodologia **BEM (Block Element Modifier)**
- Adaptar o design para diferentes resoluções de tela

---

## 🚀 Funcionalidades

- Exibição de diferentes cidades natais
- Layout responsivo para múltiplos dispositivos
- Organização visual com seções bem definidas
- Exibição de imagens e descrições de cada local
- Estrutura semântica de HTML

---

## 🛠 Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

- **HTML5** → Estrutura e semântica da página
- **CSS3** → Estilização e layout da interface
- **Flexbox / Grid** → Organização dos elementos na página
- **Metodologia BEM** → Organização e padronização das classes CSS
- **Figma** → Referência de design da interface
- **Git e GitHub** → Controle de versão e armazenamento do projeto
